Format

A sample JSON item is shown below. The acceptable tags are: 'item_name', 'count', 'model_number', 'description', 'tags', 'custom_fields'.

NOTE: 'item_name' and 'count' are required, and 'count' must be a positive integer.

Import multiple items by enclosing comma-separated JSON items in square brackets (i.e., [{item},{item},{item},...])

{

'item_name': "str",

'count': int,

'description': "str",
